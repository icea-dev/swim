Definition identifier URNs in OGC namespace (definitions 1.1.2)

OGC Document ID:  07-092r1
Document Version: 1.1.2
Document type:    OGC Best Practice
Date Published:   2007-11-14
Online Reference: http://www.opengeospatial.org/standards/bp

This Best Practices Paper specifies Universal Resource Names (URNs) in
the "ogc" URN namespace to be used for identifying definitions. This
document specifies the formats used by these URNs, plus a set of
specific URNs for specific definitions. These definitions should be used
wherever applicable by implementations of various OGC Implementation
Specifications, including GML, WMS, WFS, and WCS.
