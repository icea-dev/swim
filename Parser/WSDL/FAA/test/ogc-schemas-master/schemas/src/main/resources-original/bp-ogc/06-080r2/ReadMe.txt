OpenGIS(r) GML Application Schema For EO Products - ReadMe.txt

OGC Document ID:  06-080r2
Document Version: 0.9.0
Document type:    OGC Best Practice
Date Published:   2007-08-17
Online Reference: http://www.opengeospatial.org/standards/bp

This document defines an application schema of the Geography Markup
Language (GML) version 3.1.1 for describing Earth Observation products
(EO products) within the HMA (Heterogeneous EO Missions Accessibility)
Application Profile for the OGC(r) Catalogue Services Specification
v2.0.0 (with Corrigendum OGC 04-021r3).
