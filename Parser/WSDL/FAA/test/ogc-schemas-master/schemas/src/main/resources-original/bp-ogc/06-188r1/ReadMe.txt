GML Encoding of Discrete Coverages (interleaved pattern) (CV 0.2.1)

OGC Document ID:  06-188r1 with corrigendum 07-126 and 07-155 
Document Version: 0.2.1
Document type:    OGC Best Practice
Date Published:   2007-12-07
Online Reference: http://www.opengeospatial.org/standards/bp

A standalone, explicit GML representation of the ISO 19139 
CV_DiscreteCoverage model, following the geometry-value pair 
(interleaved) pattern.
